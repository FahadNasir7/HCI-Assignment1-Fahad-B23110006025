HTML Common Errors & Fixes

1. Missing Closing Tags

<p>This is a paragraph
<p>This is a paragraph</p>

2. Incorrect Nesting

<ul><li>Item 1<p>Text</li></ul>
<ul><li>Item 1</li></ul><p>Text</p>

3. Missing Alt Attribute in Images

<img src="image.jpg">
<img src="image.jpg" alt="Description of image">

4. Invalid Attributes

<div color="red">Text</div>
<div style="color:red;">Text</div>

5. Deprecated Tags

<font color="blue">Text</font>
<span style="color:blue;">Text</span>

6. Improper List Nesting

<ul><li>Item<ul><li>Subitem</ul></li></ul>
<ul><li>Item <ul><li>Subitem</li></ul></li></ul>

7. Incorrect Heading Order

<h3>Title</h3><h1>Main Heading</h1>
<h1>Main Heading</h1><h2>Subheading</h2>

8. Broken Links

<a href="wrong.html">Link</a>
<a href="correct.html">Link</a>

9. Inline Styles Misuse

<h1 style="font-size:20pt">Heading</h1>
<h1>Heading</h1>

10. Form Errors

<form><input type="text"></form>
<form action="submit.php"><input type="text" name="username"></form>
